# 1. Two Sum

## Problem Link
- LeetCode 1: Two Sum

## Pattern
- Hash Map
- Complement Search

## Problem Statement
Given an array `nums` and an integer `target`,
return indices of two numbers such that they add up to `target`.

---

## Brute Force Approach

### Idea
Check every pair of elements.

### Algorithm
1. Traverse array with index `i`.
2. For every `i`, check all `j > i`.
3. If `nums[i] + nums[j] == target`, return indices.

### Complexity
- Time: O(n²)
- Space: O(1)

### Code Template

```cpp
for(int i = 0; i < n; i++){
    for(int j = i + 1; j < n; j++){
        if(nums[i] + nums[j] == target){
            return {i, j};
        }
    }
}
```
